CodeCuisine - Interactive Menu System

Description: The CodeCuisine project is a simple yet dynamic web application that showcases an interactive menu system for a virtual restaurant. Built with HTML, CSS, and JavaScript, the app allows users to browse through categories of food items (e.g., Starters, Main Courses, Desserts) and create an order in real time.

Features:

  1. Dynamic Menu Display: Menu items are generated dynamically through JavaScript, making it easy to add or modify items without changing the HTML structure.
  2. Order Management: Users can select items by clicking on them, and each selection adds the item to an order list, updating the total cost with a set price per item.
  3. Responsive Layout: Styled with CSS, the layout is visually appealing, with a clean color scheme and structured sections for easy navigation.
  4. Reusable Code: JavaScript functions like displayMenuItems() and addToOrder() allow for efficient data handling, creating a modular, maintainable code structure.

Book Connect - A Refactored Digital Library System

Description: Book Connect is a structured and efficient digital library project designed to streamline the organization and display of books, authors, and genres. Through thoughtful refactoring and a modular design approach, the project enhances readability, maintainability, and reusability of code, all while implementing key software development principles.

Key Features:

  1. Object-Oriented Structure: Books, authors, and genres are represented as distinct objects, allowing for easy management and expansion of data.
  2. Abstraction and Modularity: JavaScript functions are refactored for optimal performance, resulting in reusable components that simplify future updates and modifications.
  3. Web Components Integration: The book preview feature has been transformed into a custom Web Component, enabling encapsulated, reusable elements within the web page, with plans to convert other elements for a more cohesive user experience
  4. Documentation and Testing: Detailed documentation and comprehensive testing ensure reliability, making it easier for future developers to understand and contribute to the project.
